## **Java-nba-app?**
<b>Rapid Api</b>를 사용하여 <b>NBA(National Basketball Association)</b> 관련 정보들을 제공해주는 <b>Java Application</b><br>

## <b>주요기능</b>

### [서비스 주요 기능]
- <b>NBA 정보 제공</b><br>
1. 사용자 로그인 & 회원가입 기능
2. OkHttp로 Rapid Api에 NBA정보 요청 / 응답 정보를 UI(AWT, Swing)에 표시 및 제공
3. NBA정보 검색 기능
